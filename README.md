# 🧵 ERP Básico IT'S Fabrica

Software de gestão de produção (ERP) desenvolvido para fábricas de confecção. Controla o ciclo de vida do produto, desde a referência técnica até a venda final.

## 🚀 Tecnologias Utilizadas
* **HTML5 / CSS3** (Flexbox e Variáveis CSS)
* **Vanilla JavaScript** (ES6+)
* **LocalStorage** (Persistência de dados local)
* **jsPDF** (Geração de relatórios profissionais)

## 📋 Funcionalidades
- [x] Gestão de Referências (Corte, Costura, Estoque)
- [x] Cadastro de Clientes
- [x] Controle de Pedidos e Reserva de Estoque
- [x] Exportação de relatórios em PDF
